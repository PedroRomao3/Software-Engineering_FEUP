# Product Vision

## Vision Stament
For all the lovers of gaming who don't want to play solo games and feel like the in-game lobbies are too boring.
E-LobBy allows you to create or join customized lobbies, so you can get along with other player just like you!

## Main Features
 - Create an account with your favorite games (link to your discord to be easier)
 - Pick your rank in each game (optional)
 - Lobby has: game in question, language, looking-for rank (range), nr of max players in lobby, privacy settings (if private: has password to get in).
 - User can create lobby
 - Lobby creators can close the lobby anytime
 - Filter the lobbies, and join the ones that seem good to you
 - When the lobby is full, all the players receive a link to a shared discord between them, and the lobby is closed

## Required API's
- Discord API for direct redirection and BOT maintainer for discord server
