# E-LobBy

## Vision Stament
For all the lovers of gaming who don't want to play solo games and feel like the in-game lobbies are too unfair and borring.
E-LobBy is the perfect solution for those lonely nights where you can get along with other player just like you!

## Main Features
 - Create an account with your favorite games (link to your discord to be easier)
 - Pick your rank in each game (optional)
 - Lobby has: game in question, language, looking-for rank (range), nr of max players in lobby
 - User can create lobby
 - Lobby creators can close the lobby anytime
 - Filter the lobbies, and join the ones that seems good to you
 - When the lobby is full all the player recieve a link to a shared discord between them, and the lobby is closed

## Required API's
- Discord API for direct redirection and BOT maintainer for discord server
